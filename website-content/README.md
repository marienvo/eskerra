# Website Content

Short-form copy and philosophy text for the Eskerra website. Everything here is written in US English.

## Files

| File | Purpose |
|---|---|
| `manifest.md` | The Eskerra Manifest — full principles doc, suitable for an about or philosophy page |
| `philosophy.md` | Mid-length philosophy section, suitable for a README or landing page prose block |
| `hero.md` | Hero headline, subheadline, subhero, philosophy strip, and CTA |
| `features.md` | Feature block copy — title + short summary for each key capability |
| `snippets.md` | One-liners, two-liners, and contrast phrases for banners, social, and secondary copy |

## Core themes

1. **Markdown is the source** — not an export format, not a display layer. The file is the truth.
2. **Speed is respect** — performance is a principle, not a feature to add later.
3. **Desktop and mobile have different jobs** — depth vs. capture; no forced compromise.
4. **No lock-in** — files stay portable and understandable without Eskerra installed.
5. **Calm over chaos** — no plugin dependency, no hidden complexity, strong fundamentals.
6. **Predictable by design** — no mysterious rewrites, no silent metadata, no hidden traps.
7. **Editor-first** — the editor is the center, not one widget in a dashboard carnival.
